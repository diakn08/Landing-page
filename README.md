# Landing page

Oписание проекта: 
Этот проект был создан для фотографа, как личный сайт-визитка для брони его услуг заранее. 
Основной смысл в том чтобы завлечь иностранных туристов на покупку туров и фотоссесий, красивым лаконичным дизайном и природой Кыргызстана, так как он сам фотограф.
На этом же сайте можно ознакомиться с его работами, видами туров и фотоссесий, основным графиком, приемущества и то что нужно брать с собой и многое другое.
Для удобства есть перевод языка, поэтому могут пользоваться как и туристы, так и местные 



## Функционал
- Пользователь заходит на сайт и ознакамливается с нашим заказчиком и его ближайшими турами на этот год 
- Далее идет уже более подробая информация для каждого тура, переноситься при нажатии на другое окно с полным опиманием определенного тура и уже нажимая на кнопку BOOKED, забронировать/купить тур
- Проект умеет бронировать/покупать, информировать и ознакамливать 

## Технологии
- JavaScript / HTML / CSS / React 
- Библиотеки, фреймворки
